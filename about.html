Web service load time analysis

This map represents the average page load time of our web service across different countries. Hovering over a country will display a histogram of that country's data: this allows the viewer to get a sense of the spread of the data, not just the average. Selecting two countries in succession charts an overlay of both. If the raw volumes from the two countries are very different, you can click the 'view percentages' button to normalize both histograms for easy comparison. Comparing, for example, India and China this way reveals a "fatter tail" for India: this sort of observation could guide how we approach site optimization in different locations.

The page load time data was collected from javascript, using the HTML5 navigation timing API, and then posted back to our web service. The records are stored in a database and periodically analyzed to produce a aggregate summary, which is retrieved by this web page over AJAX. The map is a free SVG resource retrieved from Wikipedia, and the choropleth ("heat map") data bindings and histograms are generated using the d3.js library.
